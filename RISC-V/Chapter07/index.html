Chapter 7#

Running Zephyr on LiteX/VexRiscv on Avalanche board with Microsemi PolarFireFPGA#

Linux 및 관련 도구는 대부분 해당 프로젝트의 상류 저장소에 이미 있습니다. 데비안 RISC-V wiki (일부 업데이트 포함)에 언급 된 바와 같이 : - binutils : 업스트림 (2.28은 RISC-V를 지원하는 첫 번째 릴리스입니다) - gcc : 업스트림 (7.1은 RISC-V를 지원하는 첫 번째 릴리스입니다) - glibc : 업스트림 (2.27은 RISC-V를 지원하는 첫 번째 릴리스입니다.) - Linux 커널 : 업스트림 (아키텍처 코어 코드는 커널 4.15로 이동했습니다. 커널 4.19는 시뮬레이션 된 시스템을 사용자 시스템으로 부팅하는 데 필요한 모든 드라이버를 포함합니다) - gdb : 마스터에서 업스트림 됨 - qemu : 업스트림 (2.12는 RISC-V를 지원하는 첫 번째 릴리스입니다)

현재 시작 안내서에 설명 된 Linux 지원 플랫폼에는 다음이 포함됩니다. - SiFive HiFive Unleashed - VexRiscv CPU가있는 LITEX SoC aMicrosemi PolarFire FPGA 또는 Renode 시뮬레이션 프레임 워크

BusyBear Linux를 실행하는 일반적인 QEMU 64 비트 RISC-V 시뮬레이션 대상이 있습니다. 자세한 정보는 데비안 및 Fedora 포트도 제공됩니다.
